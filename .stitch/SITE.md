# Site Blueprint: Nudge Immersive Platform

**Stitch Project ID:** `4183264824471181047`

## 1. Description & Goals
Nudge is an immersive, playful, and proactive AI productivity companion. The web application renders a cinematic 3D workspace on the home page and redirects to a full interactive dashboard when "Plan My Day" is triggered.

- [ ] `index` (Cinematic Hero with 3D desk, calendar panel, floating cards, 3D clock, glowing orb)
- [ ] `app` (Dashboard with Today Flow, sidebar, Focus mode, Priority Galaxy, Habits Plant, Calendar blocks, AI Coach)
